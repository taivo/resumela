  • Résuméla is intended to cut down resume editing time for users familiar with angularjs
  • To play with the included sample resume, use this web demo as is
  • To edit your own resume, clone https://github.com/taivo/resumela.git and open "index.html" as a local file. Then use the "Content" tab
  • Use the included 'sample-resume.json' to figure out the schema. It's simple and it allows easy experimentation with resume layout.
  • Modify any of the included templates or css classes to render/style your resume differently
  • Use your browser's print feature to print the resume or export to PDF. CSS rules are already set to hide all non-resume GUI elements.
